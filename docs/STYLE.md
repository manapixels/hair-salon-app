# Luxe Cuts - UI Style Guide

This document outlines the minimalist UI design principles for the Luxe Cuts web application. The goal is to create a clean, modern, and user-friendly interface with ample white space and rounded aesthetics.

## 1. Core Principles

- **Minimalism**: Every element should have a clear purpose. Avoid unnecessary clutter and decoration.
- **Whitespace**: Use generous spacing to create a sense of openness and guide the user's focus.
- **Clarity**: Ensure that the UI is intuitive and easy to navigate.
- **Consistency**: Maintain a consistent design language across all components and pages.

## 2. Color Palette

- **Primary**: `#FFFFFF` (White) - Used for backgrounds and main content areas.
- **Secondary**: `#F3F4F6` (Light Gray) - Used for secondary backgrounds and dividers.
- **Accent**: `#4F46E5` (Indigo) - Used for primary buttons, links, and active states.
- **Text**: `#111827` (Dark Gray) - Used for primary text.
- **Subtle Text**: `#6B7280` (Medium Gray) - Used for secondary text and placeholders.
- **Borders**: `#E5E7EB` (Gray) - Used for borders and dividers.

## 3. Typography

- **Font Family**: Inter (or a similar sans-serif system font).
- **Headings**:
  - `h1`: 36px, Bold
  - `h2`: 24px, Bold
  - `h3`: 18px, Semi-bold
- **Body Text**: 16px, Regular
- **Secondary Text**: 14px, Regular

## 4. Layout & Spacing

- **Grid System**: Use a flexible grid system (e.g., 12-column grid) for consistent layouts.
- **Padding**:
  - Page containers: `32px`
  - Cards and sections: `24px`
- **Margins**: Use consistent margins between elements (e.g., `16px`, `24px`, `32px`).

## 5. Components

### Buttons

- **Border Radius**: `8px` (rounded-lg)
- **Padding**: `12px 24px`
- **Primary Button**: Indigo background with white text.
- **Secondary Button**: White background with a gray border and dark gray text.
- **Hover State**: Slightly darker shade of the background color.

### Cards & Containers

- **Border Radius**: `12px` (rounded-xl)
- **Box Shadow**: A subtle shadow to create depth (e.g., `0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06)`).
- **Padding**: `24px`.

### Forms & Inputs

- **Border Radius**: `8px` (rounded-lg)
- **Padding**: `12px`
- **Border**: `1px` solid gray border.
- **Focus State**: Indigo border color.
- **Labels**: Placed above the input field, semi-bold.

### Navigation & Tabs

- **Active Tab**: Indigo text with an indigo underline.
- **Inactive Tab**: Medium gray text.
- **Spacing**: Generous spacing between tab items.
